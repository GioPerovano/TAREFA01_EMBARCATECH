# TAREFA01_EMBARCATECH

# Sistema de Controle com Teclado Matricial e Buzzer

## Instruções:

- **'A'**: Ativa o LED verde.
- **'B'**: Ativa o LED azul.
- **'C'**: Ativa o LED vermelho.
- **'D'**: Ativa todos os LEDs.
- **'#'**: Ativa o buzzer.
- **'*'**: Desliga todas as LEDS.
- **'0'**: Alterna entre todas as cores.
